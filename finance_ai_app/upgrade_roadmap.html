P1 — Kiến trúc Nền tảng & khả năng mở rộng Ưu tiên #1
Repository pattern
Tách DB query ra lớp riêng. TransactionRepo, CategoryRepo — UI không gọi SQL trực tiếp nữa
Event bus nội bộ
QObject signal trung tâm. Các frame subscribe thay vì main_window.refresh_all() gọi tất cả
Plugin system cho AI
BaseAIPlugin interface. Thêm engine mới không sửa chatbot_frame. Dễ hot-swap
Multi-user / family
Bảng user_profiles riêng. Mỗi user có data space độc lập, share được theo nhóm

P4 — Bảo mật & Errors Robustness Ưu tiên #4
DB context manager
get_connection() trả DBContext, dùng with statement. Tự commit/rollback, không leak
Global error handler
sys.excepthook bắt unhandled exception. Log file, user-friendly dialog, không crash im lặng
Data validation layer
Pydantic-style model cho Transaction, Budget. Validate trước khi lưu DB
